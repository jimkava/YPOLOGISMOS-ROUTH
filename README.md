# Υπολογιστής Routh-Hurwitz

Εφαρμογή κινητού για τον υπολογισμό της ευσταθείας συστημάτων ελέγχου με τη μέθοδο Routh-Hurwitz.

## Χαρακτηριστικά

- Υποστήριξη χαρακτηριστικών εξισώσεων από 1ου έως 6ου βαθμού
- Υπολογισμός πίνακα Routh
- Προσδιορισμός κρίσιμων τιμών (Κκρ, ωκρ)
- Ανάλυση σταθερότητας συστήματος
- Cross-platform (iOS & Android)
- Ελληνική γλώσσα

## Απαιτήσεις

- Node.js (v14 ή νεότερη)
- npm ή yarn
- Expo CLI
- Expo Go app (για testing σε πραγματική συσκευή)

## Εγκατάσταση

```bash
# Εγκατάσταση dependencies
npm install

# ή με yarn
yarn install
```

## Εκτέλεση

```bash
# Εκκίνηση του development server
npm start

# ή
npx expo start
```

Στη συνέχεια μπορείτε να:
- Σαρώσετε το QR code με την εφαρμογή Expo Go στο κινητό σας
- Πατήστε 'a' για Android emulator
- Πατήστε 'i' για iOS simulator
- Πατήστε 'w' για άνοιγμα στον browser

## Χρήση

1. Επιλέξτε τον βαθμό της χαρακτηριστικής εξίσωσης (1-6)
2. Εισάγετε τους συντελεστές της εξίσωσης
3. Εισάγετε την τιμή του K
4. Πατήστε "ΥΠΟΛΟΓΙΣΜΟΣ"
5. Δείτε τα αποτελέσματα:
   - Χαρακτηριστική εξίσωση
   - Πίνακας Routh
   - Κατάσταση σταθερότητας
   - Κρίσιμες τιμές (αν υπάρχουν)

## Μεθοδολογία Routh-Hurwitz

Η μέθοδος Routh-Hurwitz είναι ένα μαθηματικό κριτήριο που καθορίζει αν ένα σύστημα ελέγχου είναι ευσταθές εξετάζοντας τους συντελεστές της χαρακτηριστικής εξίσωσης του συστήματος.

Για μια χαρακτηριστική εξίσωση:
```
aₙsⁿ + aₙ₋₁sⁿ⁻¹ + ... + a₁s + a₀ = 0
```

Το σύστημα είναι ευσταθές αν όλα τα στοιχεία της πρώτης στήλης του πίνακα Routh έχουν το ίδιο πρόσημο.
